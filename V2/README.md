# Torque.JS V2: Classic Engine Sound Emulator

Pure HTML/CSS/JavaScript engine sound simulator - no build process required.

## Overview

Torque.JS V2 is the classic, self-contained version featuring a single HTML file with embedded CSS and JavaScript. Perfect for quick deployment and static hosting.

## Features

### Engine Support
- **V6 Engine**: 6-cylinder configuration
- **V8 Engine**: 8-cylinder configuration  
- **V10 Engine**: 10-cylinder configuration
- **V12 Engine**: 12-cylinder configuration

### Visual Features
- GIF-based engine visualizer
- Speedometer and tachometer gauges
- Real-time audio waveform display
- Classic Torque.js UI design

### Controls
- Throttle slider
- Gear shifting
- Real-time audio synthesis

## Usage

Simply open `Torque.html` in any modern web browser. No installation or build process required.

## File Structure

```
V2/
├── README.md          # This file
├── Torque.html        # Main application (single file)
├── App_Logo.png       # Application logo
├── manifest.json      # Web app manifest
├── assets/            # Engine GIFs and images
├── presets/           # Engine configuration files (.ini)
└── audio/             # Audio sample files
```

## Configuration

Engine configurations are stored in `presets/*.ini` files. Each engine can be customized with:
- Engine parameters
- Sound sources
- Visual settings

## Browser Support

- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)

Overview

Torque.JS V2 is the classic, lightweight version built with pure HTML/CSS/JavaScript. It runs offline instantly with no build, generating realistic V6/V8/V10/V12 engine sounds using pure tone synthesis.

Key Features

  • Zero dependencies, single-page app
  • Real-time synthesis, no audio samples
  • Preset system for multiple engine types
  • Temperature and slip systems for realism

Getting Started

  1. Open V2/Torque.html in a modern browser.
  2. Load an engine preset and start tweaking parameters.
  3. Use the controls to manage volume, mute, and telemetry.

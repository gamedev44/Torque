Preset Files

Preset configurations are stored as INI files in V2/presets/ (e.g., v6.ini, v8.ini, v10.ini, v12.ini).

How To Use

  1. Open V2/Torque.html.
  2. Choose an engine type and load a preset.
  3. Adjust parameters like RPM range, harmonics, and limiter characteristics.

Tips

  • Start from a nearby engine configuration, then fine-tune.
  • Use the telemetry panel to monitor RPM, temperature, and effects.

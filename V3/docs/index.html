Overview

Torque.JS V3 is the advanced TypeScript-based version with modern web APIs, video visualizers, expanded engine templates (V2–V16), and a robust INI configuration system.

Highlights

  • Real-time synthesis with advanced physics
  • Video and canvas visualizers for multiple engines
  • Search/select system and templated engine library
  • Static build for hosting or offline use

Getting Started

  1. Open V3/torque-v3/index.html (dev) or V3/torque-v3/webdist/index.html (static build).
  2. Pick an engine template and adjust parameters in the UI.
  3. Use deployment guide for hosting static assets.

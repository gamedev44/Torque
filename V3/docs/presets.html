Preset Files

Engine templates and presets live in V3/torque-v3/presets/ (e.g., v6.ini, v8.ini, v10.ini, v12.ini).

How To Use

  1. Open V3/torque-v3/index.html for dev or webdist/index.html for static.
  2. Select engine type and load or edit preset values.
  3. Use UI telemetry to validate RPM/heat while tuning.

Tips

  • Start with a close engine template then refine harmonics.
  • Review STATIC_DEPLOYMENT.md for hosting workflow.

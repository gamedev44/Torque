Static Deployment Guide

← V3 Home Docs Open Static Build

Overview

The static build is located at V3/torque-v3/webdist/. Host this directory on any static server (e.g., GitHub Pages, Netlify, Vercel) to serve the app without a dev server.

Steps

  1. Build or verify assets in webdist.
  2. Upload the entire directory to your static host.
  3. Ensure relative paths remain intact.

Notes

  • Media and presets are under V3/torque-v3/public/ and presets/.
  • For local preview, open webdist/index.html directly.
